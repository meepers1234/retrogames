The Games on this website can save. The way you save the games is by hovering over the game with your mouse and select "save state", or simply press the "F5" button on your keyboard. The website will save a file ending in the ".save" extension. When you return to the game, hover over the game and select "load state", or press the "F8" key. From there select the file you previously saved.
